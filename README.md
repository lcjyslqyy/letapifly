### 这是一个API测试工具.在这里,你只需要上传按照格式上传xml文档和设置好你的接口对应参数,你就可以保存这些信息.然后你可以转到测试接口的页面,对这个接口进行设置参数和测试.这个工具让我们接口开发人员能更简单的进行单元测试和不需要手写API文档.若安卓或IOS问你要接口,你可以把接口名称或接口链接的关键字给他自己去搜索.multiselect是支持模糊匹配的.
